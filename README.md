# Urban Brew - Комплект графики для веб-сайта кофейни

## Описание проекта

Комплект графических материалов для веб-сайта кофейни "Urban Brew", включающий айдентику, иконки, графические элементы, анимации и демо-страницу.

## Цветовая палитра

- **Основной цвет:** `#3E2723` (Темно-коричневый) - для текста и акцентов
- **Акцентный цвет:** `#8D6E63` (Средне-коричневый) - для кнопок и выделений
- **Фоновый цвет:** `#EFEBE9` (Светло-бежевый) - для фонов
- **Дополнительный цвет:** `#5D4037` (Коричневый) - для второстепенных элементов
- **Светлый цвет:** `#D7CCC8` (Светло-серый) - для текста на темном фоне

## Типографика

- **Заголовки:** Playfair Display (Google Fonts)
  - H1: 48px, font-weight: 700
  - H2: 32px, font-weight: 700
  - H3: 24px, font-weight: 500

- **Основной текст:** Roboto (Google Fonts)
  - Body: 16px, font-weight: 400
  - Small: 14px, font-weight: 400

## Логотип

Логотип "Urban Brew" состоит из:
- Символа кофейной чашки с паром
- Текста "Urban Brew" в двух цветах
- Темного фона (#3E2723)

**Форматы экспорта:**
- SVG (векторный)
- PNG 1x (200x80px)
- PNG 2x (400x160px) для Retina дисплеев

## Иконки

Набор из 11 векторных иконок в едином линейном стиле:
- Меню
- Контакты
- Кофе с собой
- Wi-Fi
- Веранда
- Кофе
- Локация
- Телефон
- Часы
- Сердце
- Дом

Все иконки экспортированы в SVG формате, размер 32x32px.

## Графические элементы

1. **Зерно кофе** - векторная иллюстрация зерна (64x64px)
2. **Чашка кофе** - векторная иллюстрация чашки с паром (80x100px)
3. **Лист** - декоративный элемент листа (60x80px)

## Паттерны

Создан текстурированный паттерн для фона сайта с элементами кофейных зерен и точек. Паттерн повторяется бесшовно.

## Баннеры для соцсетей

Макет баннера размером 1200x630px для использования в социальных сетях. Включает:
- Логотип и название
- Слоган
- Описание
- Призыв к действию
- Декоративные элементы

**Форматы экспорта:**
- HTML (для просмотра)
- JPG (1200x630px)
- WebP (1200x630px) - для лучшего сжатия

## Анимации

### 1. GIF анимация логотипа
- Циклическая анимация вращающейся чашки
- Анимация пара
- Появление текста
- Размер: 200x80px (1x), 400x160px (2x)
- Формат: GIF (оптимизированный)

### 2. Анимация баннера (Hero section)
- Появление элементов слева и справа
- Плавающая анимация чашки
- Пульсирующий фон
- Вращающиеся декоративные элементы
- Длительность: 5-10 секунд
- Формат: MP4 (видео) или Lottie JSON

## Интеграция на сайт

### GIF анимация
```html
<img src="assets/animations/logo-animation.gif" alt="Urban Brew Logo">
```

### Видео анимация
```html
<video autoplay loop muted playsinline>
    <source src="assets/animations/hero-banner.mp4" type="video/mp4">
</video>
```

### Lottie анимация
```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/bodymovin/5.12.2/lottie.min.js"></script>
<div id="lottie-container"></div>
<script>
    lottie.loadAnimation({
        container: document.getElementById('lottie-container'),
        renderer: 'svg',
        loop: true,
        autoplay: true,
        path: 'assets/animations/logo-animation.json'
    });
</script>
```

## Рекомендации по использованию

1. **Размеры файлов:**
   - Оптимизируйте все изображения перед использованием
   - Используйте WebP для лучшего сжатия
   - Предоставляйте версии 1x и 2x для Retina дисплеев

2. **Условия воспроизведения:**
   - Автовоспроизведение видео только с атрибутом `muted`
   - Учитывайте `prefers-reduced-motion` для доступности
   - Оптимизируйте размер файлов для быстрой загрузки

3. **Форматы экспорта:**
   - Логотип: SVG, PNG (1x, 2x)
   - Иконки: SVG
   - Графика: SVG
   - Баннеры: JPG, WebP
   - Анимации: GIF, MP4, JSON (Lottie)

## Технические требования

- Все векторные файлы в формате SVG
- Растровые изображения в PNG/JPG/WebP
- Анимации оптимизированы по размеру
- Все файлы готовы к использованию в веб-разработке
