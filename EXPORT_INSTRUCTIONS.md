# Инструкции по экспорту
## Логотип
- SVG: `assets/logo/urban-brew-logo.svg`
- PNG (1x): 200×80 px
- PNG (2x): 400×160 px

## Иконки
- Используйте SVG напрямую (32×32 px)

## Баннер
- Откройте `assets/banners/social-banner.html`
- Сделайте скриншот 1200×630 px
- Сохраните как JPG/WebP

## Анимации (в учебных целях)
- Для GIF: используйте CSS-анимацию → запишите экран → конвертируйте в GIF
- Для видео: используйте MP4 (H.264)

## Структура папок уже готова — просто упакуйте в ZIP.
